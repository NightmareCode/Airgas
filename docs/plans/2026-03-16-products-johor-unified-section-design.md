## Goal
Make Johor products appear as one unified section on the Products page.

## Problem
Johor products are currently presented as multiple separate sub-sections, which makes the page feel fragmented.

## Approach (Option A)
- Keep a single Johor branch container with one title.
- Replace multiple Johor sub-sections with a single grid of category cards.
- Preserve existing deep-link anchors by moving the existing IDs onto the category cards.

## Result
- One Johor section containing: Regulators, Valves, and Other categories (Fire Safety, Medical Oxygen Sets, Gas Cylinders).
